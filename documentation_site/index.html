Project Overview

The Node.js Subscription Tracker is a backend API designed to help users manage their financial commitments. It solves the problem of "subscription fatigue" by providing a centralized system to track renewal dates and costs.

Whether you are a manager looking to understand the product capabilities, or a developer joining the team, this documentation site covers everything you need to know.
